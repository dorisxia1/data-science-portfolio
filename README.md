A curated portfolio of data science projects focused on real-world data analysis, machine learning, and AI fairness, with applications in public health, databases, and interactive visualization.

## Projects

### 1. NYC EMS Fairness Analysis

Analyzed disparities in emergency response times using NYC EMS dispatch data and ACS demographic data.
Conducted severity-controlled fairness evaluation using statistical testing and regression modeling.

**Skills:** Python (pandas, numpy), statistical testing, regression, data cleaning, fairness analysis
→ Includes full paper and reproducible analysis pipeline

---

### 2. ChatDB – Natural Language Database Interface

Built an AI-powered system that translates natural language into SQL and MongoDB queries.
Implemented schema exploration and query generation for complex database operations.

**Skills:** LLM integration, SQL, MongoDB, backend logic

---

### 3. Dementia Visualization Dashboard

Developed an interactive visualization system to explore dementia prevalence and risk factors across U.S. counties.
Designed a choropleth map with a custom "Mismatch Index" to highlight regions where observed outcomes differ from medical expectations.

**Skills:** Vue.js, D3.js, data visualization, geospatial analysis
