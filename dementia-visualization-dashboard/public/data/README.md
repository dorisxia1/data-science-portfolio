# Data Folder Documentation

This folder contains all datasets used in the project, organized into two subfolders: `/raw` and `/cleaned`. Below is a description of each subfolder and the datasets they contain.

---

## `/raw`
This folder contains the raw datasets used as inputs for pre-processing scripts. These datasets are sourced from publicly available repositories and are described in the "Data Sources" section of the NestedAccordion component.

### Files:
1. **Dementia Datahub PUF 2020 Codebook.xlsx**  
   - Description: Metadata and documentation for the NORC Dementia DataHub datasets.
   - Source: NORC at the University of Chicago.

2. **NORC_Dementia_DataHub_County_2020.csv**  
   - Description: County-level dementia data for 2020, including prevalence, incidence, and mortality rates.
   - Source: NORC Dementia DataHub.

3. **NORC_Dementia_DataHub_Nat_State_2020.csv**  
   - Description: National and state-level dementia data for 2020, including demographic breakdowns.
   - Source: NORC Dementia DataHub.

4. **PLACES__Local_Data_for_Better_Health__County_Data_2024_release_20250307.csv**  
   - Description: County-level health data from the CDC PLACES dataset, including chronic disease prevalence and risk factors.
   - Source: CDC PLACES.

5. **US_County_Precipitation.csv**  
   - Description: Annual precipitation data for U.S. counties in 2020.
   - Source: National Centers for Environmental Information.

6. **US_County_Temperatures.csv**  
   - Description: Annual temperature data for U.S. counties in 2020.
   - Source: National Centers for Environmental Information.

7. **annual_aqi_by_county_2020.csv**  
   - Description: Air Quality Index (AQI) data for U.S. counties in 2020, including pollutant-specific metrics.
   - Source: U.S. Environmental Protection Agency.

8. **state-code-book.txt**  
   - Description: A text file mapping state abbreviations to full state names.

---

## `/cleaned`
This folder contains processed and cleaned datasets generated by the pre-processing scripts. These datasets are used directly in the visualizations and analyses across the project.

### Files:
1. **EnvironmentalViewData.csv**  
   - Description: Combined dataset of dementia outcomes and environmental risk factors for use in the EnvironmentalView component.
   - Generated by: `preprocess_environmental_data.py`.

2. **circlePackingData.csv**  
   - Description: Cleaned dataset for visualizing Medicare coverage patterns in the MedicareView component.
   - Generated by: `data_cleaning.ipynb`.

3. **circlePackingData_withSyntheticYears.csv**  
   - Description: Synthetic Medicare data for 2020-2024, used in the MedicareView component.
   - Generated by: `synthetic_medicare_data_builder.py`.

4. **counties.json**  
   - Description: GeoJSON file containing county boundaries for mapping visualizations.
   - Used in: RiskHotspotsView component.

5. **states.json**  
   - Description: GeoJSON file containing state boundaries for mapping visualizations.
   - Used in: RiskHotspotsView component.

6. **fips_states_codebook.csv**  
   - Description: Simplified mapping of FIPS codes to state names.
   - Generated by: `data_cleaning.ipynb`.

7. **fips_states_counties_codebook.csv**  
   - Description: Detailed mapping of FIPS codes to state and county names.
   - Generated by: `FPIS_map_meta.py`.

8. **norc_dementia_data_bar.csv**  
   - Description: Cleaned dataset for demographic bar chart visualizations in the DemographicView component.
   - Generated by: `data_cleaning.ipynb`.

9. **population.csv**  
   - Description: Population data for U.S. states, used for normalization in the MedicareView component.
   - Source: External population dataset.

10. **dementia_mismatch_index.csv**  
    - Description: Dataset containing the mismatch index for dementia prevalence, used in the RiskHotspotsView component.
    - Generated by: `dementia_mismatch_index.py`.

---

## Usage in Components
The datasets in `/cleaned` are used across various components in the project:

- **RiskHotspotsView.vue**  
  - Uses: `counties.json`, `states.json`, `dementia_mismatch_index.csv`.

- **MedicareView.vue**  
  - Uses: `circlePackingData_withSyntheticYears.csv`, `fips_states_codebook.csv`, `population.csv`.

- **EnvironmentalView.vue**  
  - Uses: `EnvironmentalViewData.csv`.

- **DemographicView/demographic_bar.vue**  
  - Uses: `norc_dementia_data_bar.csv`.
